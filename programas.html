Curso de Programación Web

Duración: 3 meses

Módulos:

  • Introducción a HTML (1 semana)
  • Estilos con CSS (1 semana)
  • JavaScript Básico (2 semanas)
  • JavaScript Avanzado (2 semanas)
  • Introducción a React (1 semana)
  • Proyecto Final (2 semanas)

Este programa está diseñado para proporcionar a los estudiantes una comprensión sólida de los fundamentos del desarrollo web, comenzando desde HTML y CSS hasta JavaScript y React, culminando en un proyecto final.
