En la Escuela DevRock ofrecemos clases de programación en HTML, CSS, JavaScript y React. Prepárate para el futuro con una educación de calidad y práctica.

Nuestros Lenguajes de Programación

HTML

HTML es el lenguaje de marcado utilizado para crear la estructura de las páginas web.

CSS

CSS es el lenguaje de estilo que se utiliza para diseñar y dar formato a las páginas web.

JavaScript

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web.

React

React es una biblioteca de JavaScript utilizada para construir interfaces de usuario.
